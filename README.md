# Multi-Language Local File Sharing 🚀

A lightweight, fast, and multi-language local network file sharing system built with Python (Flask). Transfer files seamlessly between your PC and mobile devices without internet dependency.

---

## 🌍 Supported Languages
The system dynamically loads translations from the `LANG.json` file. Supported language codes:
* `PT-BR` (Português - Brasil)
* `PT-PT` (Português - Portugal)
* `EN-US` (English - United States)
* `ES-ES` (Español - España)
* `ES-US` (Español - Estados Unidos)
* `RU-RU` (Русский - Россия)

> ⚙️ **How to change language:** Open `LANG.json` and change the `"current_language"` value to your preferred code (e.g., `"PT-BR"`).

---

## 💻 Windows (10/11)

### Requirements
* Python 3
* Wi-Fi Connection
* 2 Devices on the same network

### How to start?
1. Right-click the `iniciar.bat` file in this folder.
2. Select **Run as Administrator**.
