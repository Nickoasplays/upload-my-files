---Windows (10/11)---
Requisitos:
    Python
    Wi-Fi
    2 dispositivos
Como iniciar?
    Inicie o arquivo .bat nessa mesma pasta, e garanta que tenha executado como adiministrador!

---Linux---
Requisitos:
    Python
    Wi-Fi
    2 dispositivos
Como iniciar?
    Abra o seu terminal nessa pasta, execute "chmod +x iniciar.sh" e em seguida execute "./iniciar.sh"
    "chmod +x iniciar.sh" só precisa ser executado apenas uma única vez. O comando pode variar de distro para distro.
---AVISO!---
A pasta "upload" é para permitir que aquele(s) arquivo(s) seja(m) disponibilizado para baixar (necessário pelo menos 1 arquivo para continuar)
A pasta "download" é a pasta onde ira enviar arquivos

---Dispositivo que irá enviar/receber---

No terminal do dispositivo primário (o mesmo que está lendo esse "LEIA ISSO.txt") aparecerá um endereço IP. Use ele no dispositivo secundário para acessar o site (AMBOS OS DISPOSITIVOS TEM QUE ESTAR NA MESMA REDE WI-FI!)
Em seguida, escolha o que deseja fazer: Receber arquivos/Enviar Arquivos

---Receber arquivos---
Ao entrar nesse menu, escolha os arquivos que deseja baixar e ele começara o download.

---Enviar arquivos---
Ao entrar nesse menu, escolha os arquivos que deseja enviar e ele começara o envio
