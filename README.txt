# Local File Sharing 🚀

A lightweight and fast local network file sharing system built with Python (Flask).

---

## 💻 Windows (10/11)
### Requirements:
* Python
* Wi-Fi Connection
* 2 Devices

### How to start?
Run the `.bat` file in this folder. **Make sure to run it as Administrator!**

---

## 🐧 Linux
### Requirements:
* Python
* Wi-Fi Connection
* 2 Devices

### How to start?
1. Open your terminal inside this project folder.
2. Grant execution permission to the script:
   ```bash
   chmod +x start.sh
